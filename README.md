# 🚀 PricePeak AI

AI-powered smart shopping and price comparison platform.

PricePeak helps users:
- Compare prices across platforms
- Track historical product prices
- Predict future price drops using AI
- Discover best buying opportunities
- Analyze products visually with charts

# 📦 Installation

## 1. Extract ZIP

Unzip the project folder.

---

## 2. Open Terminal

Navigate to project:

```bash
cd frontend
```

---

## 3. Install Dependencies

```bash
npm install
```

---

## 4. Start Development Server

```bash
npm run dev
```

---

## 5. Open Browser

Visit:

```bash
http://localhost:5173
```
